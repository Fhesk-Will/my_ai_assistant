## 界面布局

打开助手后，你会看到一个分为左右两栏的界面：

```
┌─────────────────────┬──────────────────────────────────────────┐
│                     │                                          │
│   侧边栏            │   对话区域                               │
│                     │                                          │
│  ┌───────────────┐  │   ┌──────────────────────────────────┐  │
│  │  + 新对话     │  │   │  当前对话标题                    │  │
│  └───────────────┘  │   └──────────────────────────────────┘  │
│                     │                                          │
│  对话列表           │   消息区域                               │
│  ─────────────      │   （助手消息靠左，你的消息靠右）         │
│  · 对话 1           │                                          │
│  · 对话 2  ←当前    │                                          │
│  · 对话 3           │                                          │
│                     │                                          │
│  ─────────────      │   ┌──────────────────────────────────┐  │
│  监控面板           │   │  输入框          [发送]           │  │
└─────────────────────┴───┴──────────────────────────────────┴──┘
```

### 侧边栏（左侧）

| 元素 | 位置 | 作用 |
|------|------|------|
| **+ 新对话** | 顶部 | 创建一个新的对话 |
| **对话列表** | 中间 | 显示所有历史对话，点击可切换 |
| **监控面板** | 底部 | 跳转到系统监控页面 |

### 对话区域（右侧）

| 元素 | 位置 | 作用 |
|------|------|------|
| **对话标题** | 顶部 | 显示当前对话的名称 |
| **消息区域** | 中间 | 显示完整的对话历史 |
| **输入框** | 底部 | 输入你想说的内容 |
| **发送按钮** | 底部右侧 | 点击发送消息 |

## 发送消息

### 基本发送

1. 点击底部输入框，光标会出现在输入框中
2. 输入你想说的内容
3. 按 `Enter` 键，或点击右侧「**发送**」按钮

消息发送后，你的消息会出现在右侧（蓝色背景），助手的回复会出现在左侧（白色背景）。

### 输入多行内容

如果你需要换行（例如发送一段代码或多段文字），按 `Shift + Enter` 插入换行，不会触发发送。

```
单行发送：  直接按 Enter
换行输入：  按 Shift + Enter
```

### 等待回复

发送消息后，消息区域会出现「**思考中...**」的提示，表示助手正在处理你的请求。此时：

- 输入框会被禁用，无法继续输入
- 发送按钮变为不可点击状态
- 等待助手回复完成后，输入框自动恢复可用

---

## 查看回复

### 消息样式

- **你的消息**：显示在右侧，蓝色背景
- **助手的消息**：显示在左侧，白色背景带边框

### 富文本格式

助手的回复支持 Markdown 格式渲染，包括：

| 格式 | 效果 |
|------|------|
| `**加粗**` | **加粗文字** |
| `# 标题` | 大标题 |
| `` `代码` `` | 行内代码 |
| 代码块（三个反引号） | 带语法高亮的代码块 |
| `- 列表项` | 无序列表 |
| `1. 列表项` | 有序列表 |

当助手回复包含代码时，代码会以等宽字体、带背景色的代码块形式展示，便于阅读和复制。

### 自动滚动

每次收到新消息后，对话区域会自动滚动到最新消息处。如果你向上滚动查看历史记录，下次发送消息后会自动回到底部。

---

## 空白状态

当前对话没有任何消息时，消息区域会显示：

> 有什么我可以帮你的？

这是正常状态，直接在输入框输入内容开始对话即可。

---

## 输入框提示

输入框内有占位提示文字：

> 输入消息... (Enter 发送, Shift+Enter 换行)

这个提示在你开始输入后会自动消失。
