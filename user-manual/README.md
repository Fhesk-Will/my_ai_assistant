## 产品简介

**我的 AI 助手**是一个运行在本地的个人 AI 对话助手。它能记住你的偏好、跨对话积累知识，并随着使用时间的增长越来越了解你。

所有数据保存在本地，不上传到任何第三方服务器（LLM 调用除外）。

---

## 使用流程概览

```
准备环境
   ↓
启动服务（python server.py）
   ↓
浏览器打开 http://127.0.0.1:8888
   ↓
新建对话 → 输入消息 → 获得回复
   ↓
（可选）查看监控面板了解系统状态
   ↓
关闭浏览器标签页
服务端按 Ctrl+C 停止
```

## 快速参考

| 操作 | 方式 |
|------|------|
| 发送消息 | 输入后按 `Enter` |
| 换行 | `Shift + Enter` |
| 新建对话 | 点击侧边栏顶部「+ 新对话」 |
| 删除对话 | 鼠标悬停在对话上，点击「×」 |
| 打开监控面板 | 点击侧边栏底部「监控面板」 |
| 返回对话 | 点击监控面板顶部「← 返回对话」 |
